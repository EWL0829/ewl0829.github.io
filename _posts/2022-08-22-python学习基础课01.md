### python数据类型

#### 6种基本类型

- String
- Number
  - int(长整型)
  - complex(复数)
  - float(浮点数)
  - bool(True / False)
- Set
- List
- Tuple
- Dictionary

其中不可变数据类型：String Number Tuple，可变数据类型：List Dictionary Set

#### 如何判断类型

- isinstance()
- type()

区别：
isinstance是从实例的角度出发的，例如1继承自Number父类，那么1作为一个子类和Number父类是相同的一种类型.
type则不认为子类与父类是同一种类型
