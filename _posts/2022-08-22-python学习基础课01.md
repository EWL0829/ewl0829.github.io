### 具体类型定义以及相关常用操作


#### String

字符串类型，属于序列的一种

**字符串元素访问**

可用下标进行元素的访问

```python

str01 = 'string01'
print(str01[0])  // 's'

```

**字符串片段截取**

可使用头/尾下标截取

```python

str02 = 'string02'

print(str02[0:2])  // 'st'

print(str02[1:2])  // 't'

print(str02[2:2])  // ''

print(str02[3:2])  // '' 神奇的是这里并不会报错，会静默返回空字符串片段

print(str02[0:-1])  // 'string0' -1作为尾下标时，表示倒数第一个，即截取至倒数第一个元素的前一个元素，最后一个是2，这里就截取到0为止

print(str02[0: -2])  // 'string'

print(str02[:])  // 'string02'

print(str02[-1:])  // '2'  从倒数第1个开始截取，一直到最后

print(str02[-2:])  // '02' 从倒数第2个开始截取，一直到最后

print(str02[:-1])  // 'string0' 等同于str02[0:-1]的写法

print(str02[:2])  // 'st' 等同于str02[0:2]

```

**字符串的拼接**

```python

str04 = 'string04'

str05 = str04[:7] + '5'  // 'string05'

```

#### 如何判断类型

- isinstance()

type则不认为子类与父类是同一种类型
